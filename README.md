# Harjutamine
Saan edaspidi kasutada GitHubi oma tööde salvestamiseks.

GitHub Inc. asutati 2008. aastal ning selle asutajateks olid Tom Preston-Werner, Chris Wanstrath ja PJ Hyett. GitHubi teenus tehti avalikkusele kättesaadavaks samuti 2008. aastal, kui hakati pakkuma veebipõhist platvormi Git-versioonihalduse kasutamiseks. Platvorm saavutas kiiresti populaarsuse ning kasvas üheks olulisemaks tööriistaks tarkvaraarendajatele üle maailma.
